# Microsoft 365 Copilot Onboarding Guide

## Purpose

This guide helps employees understand how to begin using Microsoft 365 Copilot responsibly and effectively.

## What Copilot Can Help With

Copilot can assist with:

- Summarizing documents
- Drafting emails
- Creating meeting summaries
- Organizing notes
- Generating first drafts
- Brainstorming ideas
- Finding information from approved work content

## What Employees Should Check First

Before using Copilot, employees should confirm:

- They are using an approved company account
- They understand what information is sensitive
- They know Copilot uses existing Microsoft 365 permissions
- They do not enter confidential, personal, legal, financial, or protected data unless approved
- They review all AI-generated content before sending or publishing it

## Basic Prompt Examples

Good prompt:

> Summarize this meeting transcript into action items, owners, and deadlines.

Better prompt:

> Summarize this meeting transcript for a non-technical manager. Include action items, owners, deadlines, risks, and unresolved questions.

## Employee Checklist

- Understand approved-use scenarios
- Protect sensitive information
- Review AI output before use
- Report issues to IT support
- Attend office hours if unsure
