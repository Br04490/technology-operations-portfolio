# Copilot Troubleshooting Workflows

## Issue: User Cannot Access Copilot

Steps:

1. Confirm the user is signed into the correct Microsoft 365 account.
2. Confirm the user has the required license.
3. Confirm the user is using a supported application.
4. Check whether the issue affects one app or multiple apps.
5. Escalate to Microsoft 365 admin support if needed.

## Issue: Copilot Shows Unexpected Data

Steps:

1. Ask the user where the content appeared.
2. Identify the source file, chat, email, or document.
3. Review permissions on the source content.
4. Confirm whether the user should have access.
5. Adjust permissions if approved.
6. Escalate potential data exposure concerns.

## Issue: Copilot Gives Poor Output

Steps:

1. Review the prompt.
2. Add more context.
3. Specify the desired format.
4. Ask Copilot to revise the response.
5. Remind the user to verify output before use.

## Issue: Employee Unsure If Use Case Is Approved

Steps:

1. Document the use case.
2. Identify the type of data involved.
3. Compare against responsible-use policy.
4. Escalate to manager, IT, compliance, or security if needed.
