# Copilot Office Hours Training Plan

## Purpose

This training plan supports employees who need help learning how to use Microsoft 365 Copilot and approved AI tools responsibly.

## Audience

- New users
- Managers
- Administrative staff
- Support teams
- Employees with limited technical experience

## Session Agenda

### 1. Welcome and Purpose

Explain why the organization is adopting AI tools and how employees should use them responsibly.

### 2. Copilot Basics

Cover:

- What Copilot does
- Where Copilot appears
- How permissions work
- Common use cases

### 3. Responsible Use

Discuss:

- Sensitive information
- Data protection
- Human review
- Approved and restricted uses

### 4. Prompt Practice

Employees practice prompts for:

- Email drafting
- Meeting summaries
- Task lists
- Document summaries

### 5. Troubleshooting

Review common issues and where to get help.

## Success Measures

- Reduced basic support tickets
- Increased employee confidence
- Improved prompt quality
- Fewer risky AI-use questions
- Better adoption across departments
